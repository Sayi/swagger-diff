Changelog

What's New

  1. POST/pet Add a new pet to the store
  2. PUT/pet Update an existing pet
  3. GET/pet/findByStatus Finds Pets by status
  4. GET/pet/findByTags Finds Pets by tags
  5. POST/pet/{petId} Updates a pet in the store with form data
  6. DELETE/pet/{petId} Deletes a pet
  7. GET/pet/{petId} Find pet by ID
  8. POST/pet/{petId}/uploadImage uploads an image
  9. GET/store/inventory Returns pet inventories by status
  10. POST/store/order Place an order for a pet
  11. DELETE/store/order/{orderId} Delete purchase order by ID
  12. GET/store/order/{orderId} Find purchase order by ID
  13. POST/user Create user
  14. POST/user/createWithArray Creates list of users with given input array
  15. POST/user/createWithList Creates list of users with given input array
  16. GET/user/login Logs user into the system
  17. GET/user/logout Logs out current logged in user session
  18. PUT/user/{username} Updated user
  19. DELETE/user/{username} Delete user
  20. GET/user/{username} Get user by user name
